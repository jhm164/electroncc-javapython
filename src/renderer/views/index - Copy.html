LxYs, DwYw, Dpm Draw

A project to work with different kind of thing including the work related to design of HTML pages using visual builder to getting it as HTML file and PDF both.


You can find brief description of each of these modules below one by one.

HTML module

This module includes many sub modules including Table, Figure, Text, Equation etc.

LaTeX module

This module is to work with producing LaTeX documents and compiling them into a PDF.

Computational code module

The main task of this module is to contain code related to R/Python/MATLAB/Julia/Octave etc.

Email manager module

This is module is used to create email tasks, it includes almost all basic features of a simple email sending task.

Social media module

This module allows user to publish/post the created contents like PDFs (to Facebook, Linkedin etc.), videos (to YouTube)

Publication module

This module allows user to create publications etc. where user can design and create publications

Video presentation module

This module allows to create video created by merging and combining different kind of media like PPT, Beamer, mp3, text, mp4 etc.
